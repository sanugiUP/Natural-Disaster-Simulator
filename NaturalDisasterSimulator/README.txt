README
======

HOW TO COMPILE : on the command line type " ./gradlew run --args fileName " where fileName is the name of the input file. Or, on the command
		     line just type " ./gradlew run " and enter the file name when prompted. 

TEST INPUT FILE : Test input file included in the submission is called " input.txt " and the ResponderCommImpl class is also modified to test 
                  the data included in these entries.  

UML DIAGRAMS : UML Class Diagram and three other UML State Diagrams are included in a folder named Diagrams along with the submssion. 

RESPONSE TO CRITERIA : It is a text file name criteria.txt which is also included in the submission main folder (not in any sub-directory).
